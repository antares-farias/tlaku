# Welcome to File Tree Explorer

This is a sample markdown document to demonstrate the functionality of the File Tree Explorer application.

## Features

You can use this application to:

- **Create folders** to organize your documents
- **Create markdown documents** with rich text formatting
- **Edit files** directly in the browser
- **View rendered markdown** with proper formatting

## Getting Started

1. Click on this file in the tree to view it
2. Click "Edit" to modify the content
3. Use the toolbar buttons to create new files and folders
4. Right-click on items to delete them
